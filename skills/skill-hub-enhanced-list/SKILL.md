# SKILL.md - 增强版技能列表

**触发词**: 列出技能, 技能列表, 查看技能, 搜索技能, 技能分类, 技能搜索

**描述**: 智能技能列表系统，支持分类、分页、搜索和过滤，解决技能列表过长问题

**作者**: 小怪 (openclaw-001)

**版本**: 2.0.0

**依赖**: curl, jq (推荐)

## 功能特性

### 📊 **智能分类**
- **自动分类**: 基于技能描述和触发词自动分类
- **手动标签**: 技能作者可添加自定义标签
- **多级分类**: 大类 → 子类 → 标签体系
- **热门分类**: 显示最常用的分类

### 📄 **分页显示**
- **默认分页**: 每页20个技能
- **可配置**: 支持10/20/50/100每页
- **总数显示**: 显示总技能数和当前页信息
- **导航提示**: "显示第1-20个技能，共156个"

### 🔍 **高级搜索**
- **关键词搜索**: 名称、描述、作者、标签
- **分类过滤**: 按分类筛选技能
- **标签过滤**: 按标签筛选技能
- **依赖过滤**: 按依赖关系筛选
- **评分排序**: 按评分、下载量、更新时间排序

### 🎯 **交互体验**
- **分页导航**: 上一页/下一页/跳转到页
- **分类菜单**: 交互式分类选择
- **搜索建议**: 输入时显示搜索建议
- **结果统计**: 显示搜索/过滤结果统计

## 使用方法

### 基本列表
```
列出技能                    # 分页显示，每页20个
```

### 指定数量
```
列出技能 -n 50             # 每页显示50个
列出技能 --all             # 显示所有技能（无分页）
```

### 分类查看
```
列出技能 --category 工具    # 查看"工具"分类的技能
列出技能 --category 天气    # 查看天气相关技能
```

### 搜索技能
```
搜索技能 天气              # 搜索包含"天气"的技能
搜索技能 --tag 自动化      # 搜索"自动化"标签的技能
```

### 高级过滤
```
列出技能 --sort rating     # 按评分排序
列出技能 --filter dependency=curl  # 过滤需要curl依赖的技能
列出技能 --author 小怪     # 查看特定作者的技能
```

## 分类体系

### 预设分类
1. **工具类** (`tools`)
   - 系统工具、文件管理、网络工具
   - 标签: `system`, `file`, `network`, `utility`

2. **开发类** (`development`)
   - 编程、调试、版本控制、部署
   - 标签: `programming`, `debug`, `git`, `deploy`

3. **生活类** (`life`)
   - 天气、日历、提醒、健康
   - 标签: `weather`, `calendar`, `reminder`, `health`

4. **娱乐类** (`entertainment`)
   - 游戏、音乐、视频、社交
   - 标签: `game`, `music`, `video`, `social`

5. **学习类** (`learning`)
   - 教育、文档、翻译、研究
   - 标签: `education`, `document`, `translation`, `research`

6. **办公类** (`office`)
   - 邮件、文档、会议、项目管理
   - 标签: `email`, `document`, `meeting`, `project`

### 自动分类规则
- **关键词匹配**: 根据技能描述中的关键词自动分类
- **触发词分析**: 分析触发词确定技能类型
- **依赖推断**: 根据依赖推断技能类别
- **作者偏好**: 记录作者常用分类

    "max_pages": 5,

    "show_stats": true,
    "interactive": true

  "search": {
    "enable_fuzzy": true,
    "min_score": 0.3,
    "max_suggestions": 5

  }
